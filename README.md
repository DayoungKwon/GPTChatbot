# KoGPT-Chatbot
KoGPT2 기반 일상 대화 및 지식 대화 챗봇

# Base model
https://github.com/SKT-AI/KoGPT2

# Dataset 
한국어 챗봇 트레이닝용 문답 페어 데이터 

## Format 
Question,Answer,Category 

Category : 
일상 0, 
이별/분노/슬픔 등(부정) 1, 
사랑/기쁨(긍정) 2,
지식 3


## Dataset 
1. 일상, 이별, 사랑 데이터 
https://github.com/songys/Chatbot_data

2. AI hub 주제별 텍스트 일상 대화 데이터

3. AI hub 감성 대호 말뭉치 

4. Youtube 자막
